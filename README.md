## 📖 About

**Elite Social Club** is a modern, responsive web application for cricket enthusiasts. It provides:

- 🏏 **Live Match Updates** — Real-time scores with a pulsing live indicator
- 📊 **Match Analytics** — Detailed stats and match summaries
- 🏆 **Tournament Tracking** — Follow your favorite tournaments
- 👥 **Team Profiles** — Explore team rosters and performance data

## 🚀 Features

- ✅ Fully responsive design (Mobile, Tablet, Desktop)
- ✅ Live match cards with real-time status badges
- ✅ Smooth scrolling news ticker
- ✅ Hamburger menu for mobile devices
- ✅ Modular component-based architecture
- ✅ Clean, modern UI with Inter font
- ✅ Hover animations and micro-interactions
- ✅ Auto-closing mobile menu (click outside, ESC key, resize)
