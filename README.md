# 🏏 Elite Social Club

A modern, high-performance cricket analytics and social club website. Designed for speed, responsiveness, and a premium user experience.

## ✨ Features

- **📱 Full Responsive Design:** Optimized for Mobile, Tablet, and Desktop.
- **🔄 Component-Based Architecture:** Uses JavaScript to load headers, footers, and cards dynamically.
- **📢 Smart Notice Ticker:** A top-bar announcement system that pauses on hover.
- **🏆 Premium Match Cards:** Detailed scorecards with live pulse animations.
- **⚡ Vanilla Performance:** Built with pure HTML, CSS, and JS (No heavy frameworks).

## 📂 Project Structure

- `index.html` - Main entry point.
- `/css` - Contains `style.css`, `navbar.css`, and `footer.css`.
- `/components` - Contains reusable HTML fragments (Navbar, Cards, Ticker).
- `/img` - Project assets and logos.

## 🚀 How to Run This Project

**IMPORTANT:** Because this project uses the JavaScript `fetch()` API to load components, you cannot open `index.html` directly from your file folder.

1.  **Open the project** in VS Code.
2.  **Install the "Live Server" extension** (by Ritwick Dey).
3.  Right-click on `index.html` and select **"Open with Live Server"**.
4.  The site will run at `http://127.0.0.1:5500`.
